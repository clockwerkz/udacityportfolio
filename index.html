Bugs -N- Gems Game

This is a classic Frogger clone game written using a basic JavaScript game engine. The only victory condition was for the player to reach the top of the board, and the lose condition is being hit by a bug.
I added additional elements such as levels, rocks as obstacles that randomly spawn in, gems to collect, and a countdown timer that will end the game.
