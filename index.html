Memory Matching Game

The second project is a Memory Matching Game that's written in JavaScript. For the project, we were given the board and the font icons, and were required to build the functionality. Features include a timer, a star rating based on number of guesses, and restart/replay button.

Bugs -N- Gems Game

Creating a classic Frogger clone game using a basic JavaScript engine was our third project. The only victory condition was for the player to reach the top of the board, and the lose condition is being hit by a bug.
I added additional elements such as levels, rocks as obstacles that randomly spawn in, gems to collect, and a countdown timer that will end the game.

FeedReader Testing

The fourth project served as an introduction into Unit Testing using Jasmine. Through the classes, we learned how to use describe blocks and it/expects to test different values and expectations. We also learned how to do asyncronous testing.

My Reads React Project

The sixth project was our first React assignment. The assignment involved a bookreads API that keeps track of the status of books in our inventory. We were responsible for using React to keep track of the status of our book inventory. We also made use of React Routing to add a search page to find more books to add to the library.
